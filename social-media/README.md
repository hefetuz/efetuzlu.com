# Hizli Sosyal Medya Sistemi

Bu klasor, Efe Tuzlu icin is/firsat odakli hizli sosyal medya uretim sistemidir. Ana kanal LinkedIn, destek kanali Instagram, vitrin kanali Behance olarak kurgulandi.

## Haftalik Ritim

- Pazartesi: Product design / UX breakdown
- Sali: Kisa gorsel vitrin
- Carsamba: Case icgorusu
- Persembe: Motion, web veya brand detay
- Cuma: Haftalik ogrenim veya work in progress notu

Hedef tempo:

- LinkedIn: Haftada 5 post
- Instagram: Haftada 3 post
- Behance: Ayda 2 case update

## Calisma Kurali

1. `14-day-calendar.csv` dosyasindan siradaki postu sec.
2. `post-bank.md` icindeki ilgili post ID'sini ac.
3. Asset satirindaki gorsel/video ile paylas.
4. LinkedIn icin uzun metni, Instagram icin kisa caption'i kullan.
5. Paylasimdan 24-48 saat sonra `metrics-tracker.csv` dosyasina performansi gir.

## Basari Metrikleri

Oncelik sirasi:

1. DM veya is/freelance gorusmesi
2. Nitelikli baglanti istegi
3. Profil goruntulenmesi
4. Portfolio tiklamasi
5. Kaydetme, yorum, paylasim

## Hizli Uretim Formulu

Her mikro post su yapida olmali:

- Hook: Tek net fikir
- Context: Hangi projede, hangi problem
- Decision: Ne karar verdin
- Result: Neyi daha net, guvenilir veya kullanilir hale getirdi
- CTA: Case linki veya sohbet daveti
